# 🍫 Chocolate Sales Analysis using Microsoft Excel

## 📌 Project Overview

This project demonstrates an end-to-end sales analysis workflow using Microsoft Excel. The dataset was cleaned, transformed, analyzed, and visualized to generate meaningful business insights through an interactive dashboard.

The objective was to, me practice the real-world data analysis techniques commonly used by Data Analysts.

---

## 📊 Project Objectives

- Clean raw sales data
- Standardize and organize the dataset
- Create calculated columns
- Analyze sales performance
- Build KPIs
- Design an interactive dashboard

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
- Conditional Formatting

---

## 📂 Dataset

The dataset contains chocolate sales transactions including:

- Sales Person
- Country
- Product
- Date
- Revenue
- Boxes Shipped
- Unit Price
- Quarters
- Month
- Year
